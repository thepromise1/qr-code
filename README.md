### What I learned

During this project, I focused on perfecting the "centered card" layout. I reinforced my understanding of how to use `min-height: 100vh` on the body to ensure the content stays vertically centered regardless of screen size.

```css
body{
    font-family: "Outfit", sans-serif;
    background-color: hsl(212, 45%, 89%);
    min-height: 100vh;
   
}

```
